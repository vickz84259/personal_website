Slick

A collection of messenger bots

Slick started out as a project with the purpose of being able to send messages from Telegram to Whatsapp and vice versa.
After several months, the project shifted course to a new path.

Now it will comprise of several bots. The bots will work in Telegram, Facebook Messenger, Slack and Twitter.

About Slick

Slick started out as a project name for a service I was workin on. Slick was meant to enable someone to send messages from Whatsapp to Telegram and vice versa. I hit a roadblock due to whatsapp's stance against using third-party apis. I didn't want to invest time in it if I'd constantly be implementing features to avoid having my numbers blocked. So, now Slick acts as the project name for a family of messaging bots I'm introducing. The first one will soon launch into private beta. Send me a message if you'd like to participate.
